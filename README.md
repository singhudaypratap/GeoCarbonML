# GeoCarbonML

> Explainable Multi-Track Machine Learning for Sustainable Geopolymer Concrete using SHAP Interpretability and Pareto Optimisation

---

## Overview

GeoCarbonML is an explainable artificial intelligence (XAI) framework for predicting:

- Compressive Strength (CS) of geopolymer and alkali-activated concrete
- Embodied CO₂ emissions
- Sustainable mix-design trade-offs using Pareto optimisation

The repository combines:
- machine learning,
- explainable AI (SHAP),
- uncertainty quantification,
- sustainability analysis,
- and multi-objective optimisation

for next-generation low-carbon concrete systems.

---

## Key Highlights

- Multi-track ML framework for GPC and AAC
- Seven benchmarked ML algorithms
- Optuna-tuned XGBoost models
- SHAP explainability:
  - feature importance
  - beeswarm plots
  - dependence plots
  - interaction effects
- Bootstrap uncertainty analysis
- External validation
- Pareto optimisation for sustainable concrete design
- Publication-grade visualisations
